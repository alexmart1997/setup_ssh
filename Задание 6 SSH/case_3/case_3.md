# Кейс 3: Копирование файлов с помощью SCP

## Описание задания
Копирование файлов между локальной машиной и удаленным сервером с помощью SCP.

## Требования к результату
- Файл успешно скопирован на удаленный сервер
- Файл успешно скопирован обратно на локальную машину

## Выполнение задания

### 1. Подготовка тестового файла

```bash
echo "Это тестовый файл для SCP копирования" > test_file.txt
cat test_file.txt
```

### 2. Копирование файла на удаленный сервер

```bash
scp test_file.txt osboxes@127.0.0.1:/home/osboxes/test_file_remote.txt
```

### 3. Проверка файла на удаленном сервере

```bash
ssh osboxes@127.0.0.1 "cat /home/osboxes/test_file_remote.txt"
```

### 4. Копирование файла обратно на локальную машину

```bash
scp osboxes@127.0.0.1:/home/osboxes/test_file_remote.txt test_file_back.txt
```

### 5. Проверка скопированного файла

```bash
cat test_file_back.txt
```
